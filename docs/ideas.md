# Ideas & Future Features

## Resources

- OBS feed integration as a live video source

## DJ / Audio Sync

- Pro DJ Link integration (Rekordbox/CDJ direct sync via network)
- Timecoded vinyl signal decoding for positional sync
- Rekordbox, Serato, Traktor support via MIDI and/or Pro DJ Link

## Shapes

- Custom/complex shapes (e.g. car, logo outlines) — import SVG paths or freehand draw

## Groups

- Group keyframes on timeline — capture all member shapes' states at once
- Morph/easing between group keyframes animates all members together
- Group timeline track alongside individual shape tracks

## Output

- Record / export projector output to video file

## Effects

- More video effects beyond blur, glow, color correction, distortion, glitch
- Custom shader support for user-defined effects
